# 一、添加游戏

软件提供三种方式添加游戏。前两种需要配合 source_url.txt 文件（内容为游戏对应的帖子链接），第三种是直接添加本地文件夹（信息较少）。

## 方式一：内置解析器站点下载的游戏

支持站点：ACG嘤嘤怪、飞雪ACG、维咔ACG。

1. 在游戏目录下新建一个文本文件，命名为 source_url.txt

2. 在文件中填入该游戏对应的帖子链接，例如：

https://www.vikacg.cc/p/567658



![image1.png](images/image1.png)

3. 将 source_url.txt放在游戏启动文件（.exe）的同级目录下：



![image2.png](images/image2.png)

4. 回到软件，进入「刮削」页面 → 扫描游戏库 → 刮削元数据即可正常入库

## 方式二：其他 ACG 站点

除了内置站点外，你也可以添加其他 ACG 站点的游戏，但需要先配置自定义解析器。

准备工作：

1. 在游戏目录下同样创建 source_url.txt

2. 在软件内点击「设置」 → 「自定义解析器」 → 「添加站点」，弹出页面如下



![image3.png](images/image3.png)

Step 1 - 填写域名：

打开帖子页面，复制浏览器地址栏中 http:// 后面的域名部分（如 acgcu.com），粘贴到自定义解析器的「域名」栏。



![image4.png](images/image4.png)

Step 2 - 获取 Cookie：

在帖子页面按 F12 打开开发者工具 → 点击「网络」→ 点击「文档」(或者是document) → 刷新页面。在请求列表中点击任意一条请求，复制其 Cookie 值，粘贴到软件的 Cookie 栏。



![image5.png](images/image5.png)



![image6.png](images/image6.png)

Step 3 - 获取标题 XPath：

点击开发者工具左上角的「选择元素」按钮，然后点击帖子标题。在右侧代码区域找到高亮（灰色）的那段代码，右键 → 复制 → 复制完整 XPath，粘贴到「标题 XPath」栏。



![image7.png](images/image7.png)

Step 4 - 获取介绍 XPath：

用同样方式选中游戏介绍内容（注意蓝色框要覆盖全部介绍文字，而非仅一小段），右键灰色代码 → 复制 → 复制完整 XPath，粘贴到「内容/介绍 XPath」栏。



![image8.png](images/image8.png)

Step 5 - 保存：

填写完成后保存即可。同一站点只需配置一次，后续无需重复添加。



![image9.png](images/image9.png)

提示：配置好自定义解析器后，回到刮削中心即可正常刮削该站点的游戏信息。

## 方式三：直接添加本地游戏

如果不想配置以上内容，或者自定义解析器无法使用，可以直接添加本地文件夹。

1. 进入软件的 「游戏」页面

2. 点击右上角的 「+」按钮

3. 浏览并选择游戏所在的文件夹，点击确认



![image10.png](images/image10.png)



![image11.png](images/image11.png)

注意：软件会将所选文件夹下的每个子文件夹视为一个游戏。例如 Test 文件夹下有 Game1、Game2 两个子文件夹，需要选择 Test 文件夹才能同时添加这两个游戏。

优化显示：

导入本地游戏后，在游戏详情页点击右上角「管理图片」可以导入本地图片或网络图片



![image12.png](images/image12.png)

随后可以自己「编辑」信息，简介内部也能「插入」你所添加的图片



![image13.png](images/image13.png)

效果展示：



![image14.png](images/image14.png)

快速导入技巧：

导入本地游戏后，在游戏详情页点击右上角「编辑」可以修改来源链接。如果填入的链接域名匹配自定义解析器或内置站点，回到刮削中心即可重新刮削。这样可以跳过手动创建 source_url.txt 的步骤。

# 二、检查游戏更新

游戏需要有明确的标题和版本号才能检查更新。

1. 进入游戏详情页

2. 找到版本号旁边的更新按钮，点击即可检查



![image21.png](images/image21.png)

注意：检查结果不一定准确，请自行判断。检查期间请不要切换页面，否则可能无法收到反馈。

# 三、分类管理

软件在刮削时会自动生成标签，但自动标签不一定符合你的需求。你可以通过「系列」功能自定义分类。



![image22.png](images/image22.png)

创建自定义系列：

1. 切换到 「系列」

2. 点击右上角的 「+」按钮



![image23.png](images/image23.png)

将游戏移入系列：

1. 回到 「游戏」页面

2. 勾选想要归类的游戏

3. 右键点击游戏 → 选择 「移入自定义系列」

4. 在弹窗中选择目标系列，点击确定



![image24.png](images/image24.png)



![image25.png](images/image25.png)

# 四、管理存档

要使用存档扫描功能，需要先标记游戏为「已玩」状态（右键菜单的「添加游玩次数」或游戏详情页的「开始游玩」），并且本地确实有玩过该游戏。



![image26.png](images/image26.png)

1. 切换到 「已玩」页面

2. 点击右上角的「扫描存档位置」按钮



![image27.png](images/image27.png)

3. 扫描完成后，进入游戏详情页会出现 「存档」按钮



![image28.png](images/image28.png)

注意：source_url.txt 必须和游戏启动文件（.exe）放在同一目录下，软件通过 exe 文件名来匹配游戏。存档扫描不保证 100% 成功，如果失败可以手动填写存档路径，之后也能通过软件快速打开存档位置。



![image29.png](images/image29.png)

在游戏的根目录下将想要的存档重命名为「存档」两个字，这样软件会自动识别并导入



![image30.png](images/image30.png)

回到软件点击「存档管理」，可以看到自动导入的「导入存档」，并且能够进行「备份」「恢复」「上传至云端」（上传至云端必须在设置填写webdav）等一系列操作



![image31.png](images/image31.png)
