# Changelog

## v1.3.9 (2026-06-24)

### ✨ 新功能

- **2DFan 下载存档**
  - 存档管理对话框新增"下载存档"按钮，支持从 2DFan 论坛搜索并下载存档
  - 搜索关键词优先使用游戏启动器名，无结果再用 metadata title 分词回退，命中即停
  - 搜索结果弹窗展示，用户手动选择后下载
  - 下载的 RAR 自动解压（调用系统 WinRAR）并重打包为 ZIP 存入备份目录
  - 临时文件自动清理，无资源泄漏

- **2DFan 域名自动检测**
  - 设置页面"论坛自定义域名"新增 2DFan 配置项，默认域名 `fan2d.top`，兜底域名 `2dfan.com`
  - 输入框右侧新增"检测可用域名"按钮
  - 自动访问 `/domain` 页面，正则匹配包含"新增"或"中转"的域名，依次检测可用性
  - 检测到可用域名后自动写入并保存

- **多目录游戏库**
  - 游戏库目录支持配置多个路径，添加/删除操作
  - 每个游戏库可独立设置对应的整理目录
  - 刮削后游戏只移动到所属库的整理目录，不会跨库移动
  - 未设置整理目录的游戏库，刮削后不移动游戏
  - 扫描时遍历所有游戏库目录和整理目录
  - 旧版单路径格式自动迁移为新格式，无感升级

### 🔧 优化

- 存档管理按钮文本精简（"打开存档文件夹"→"打开存档"等）
- 刮削扫描和游戏刷新按钮均支持多目录遍历
- 已通关游戏的备份目录查找适配多整理目录

### 🐛 修复

- 修复 2DFan 域名检测误匹配导航栏等无关元素的问题
- 修复域名检测因已存域名提前返回而不执行检测的问题
- 修复刮削页面传入 JSON 数组字符串作为路径导致的 PathNotFoundException
- 修复游戏详情页备份重命名仅查找单一整理目录的问题
- **窗口关闭崩溃修复**
  - 修复关闭软件时 `flutter_windows.dll` 崩溃的问题（STATUS_STACK_BUFFER_OVERRUN）
  - 使用 `exit(0)` 替代 `windowManager.destroy()` 关闭应用，避免 Flutter 引擎状态不一致
  - 在 `onWindowClose` 中同步保存窗口设置后再退出
- **配置保存优化**
  - 新增 `AppSettings.setValues()` 批量保存方法，避免多次文件写入
  - 新增 `AppSettings.setValuesSync()` 同步写入方法，确保窗口关闭前数据完整保存
- **黑名单路径管理修复**
  - 修复黑名单路径删除后无法重新添加的问题
  - 黑名单比较时进行路径规范化（统一正斜杠/反斜杠、转小写），避免路径格式不一致导致匹配失败
  - 修复 `_removePath` 方法中异步保存未等待完成的问题
- **游戏启动修复**
  - 修复路径包含特殊字符（括号、中文、日文等）时游戏启动失败的问题
  - 添加 `_launchExe` 辅助函数，使用 exe 文件所在文件夹作为工作目录
  - 解决 `ProcessException: 目录名称无效` 错误
  - 确保游戏程序能正确找到资源文件（游戏依赖工作目录定位资源）

### ✨ 新功能

  - 双击启动工具，右键菜单支持：启动、编辑名称、修改路径、删除

  - 设置中新增"双击启动游戏"开关（位于忽略文件夹下方，网络代理设置上方）

  - 输入地址时点击"测试连接"测试该地址
  - 不输入则默认测试 feixueacg.org

- **分页状态持久化**
  - 各页面（游戏、已玩、通关、收藏）的分页浏览位置独立记忆
  - 切换到其他页面后返回，自动恢复到上次浏览的页码
  - 状态持久化到 SharedPreferences，重启后仍然有效

### 🔧 优化

- 数据库迁移 v4→v5：新增 tools 表和 games.game_launcher / games.launcher_locked 字段

- **游戏库与已玩分离**：已玩次数 ≥ 1 的游戏不再显示在"游戏库"页面，仅在"已玩"页面展示
- **游戏评分与评论系统**：为"已玩"和"已通关"页面添加评分评论功能
  - 右键菜单新增"评论"按钮，支持 0-5 星半星评分和文字评论
  - 海报卡片左下角显示星星评分图标
  - 高评分游戏在已玩/已通关页面优先排列
  - 游戏详情页版本号旁显示星星评分
  - 红色评论按钮：悬停预览评论内容、点击查看/编辑详情、双击复制评论内容
  - 评论详情窗口支持直接编辑评分和评论，支持删除评论
  - 已通关仅备份游戏支持评分评论

### 🔧 优化

- 移除游戏页面"最近游玩"排序选项（已玩游戏不再显示）
- 修复"添加时间"排序逻辑，更新游戏时保留原添加时间而非重置为当前时间
- toast 字号增大至 16，居中显示不受文本长度影响
- 编辑游戏时同步更新 metadata.json 文件，确保标题等信息一致性
- 游戏详情页回退为白色背景，移除毛玻璃效果
- 游戏详情页内容区文章图片最大宽度从 600 调整为 800

### 🐛 修复

- 修复 MouseTracker 断言错误，星星评分改用 Listener 替代 GestureDetector
- 修复半星坐标计算问题，移除不兼容的 IntrinsicWidth + LayoutBuilder
- 修复 showGlassToast 的 AnimationController vsync 崩溃，改用 TweenAnimationBuilder
- 修复 Tooltip 导致的 MouseTracker 断言错误，改用自定义 HoverReviewButton + Overlay
- 修复备份游戏评分从 DB 加载时丢失 rating/review 字段的问题
- 修复解压码未写入 downloadUrl 导致游戏详情页不显示解压码按钮的问题
