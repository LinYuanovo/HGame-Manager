# Changelog

## v1.0.4 (2026-05-29)

### ✨ 新功能
- **已通关功能**：支持标记/取消标记游戏为已通关
  - 自动备份游戏数据到 `Sorted/Cleared/Backup/` 目录
  - 智能显示：优先显示本地存在的游戏，备份仅在本地游戏删除后显示
  - 分类页面"标签"tab固定显示"已通关"标签，显示游戏数量
  - 右键菜单"标记已通关"/"取消标记已通关"
- **搜索优化**：
  - 支持搜索标题、路径、简介
  - 搜索词持久化（切换页面后保留）
  - 搜索时保存/恢复页码
  - 搜索框添加清除按钮

### 🔧 优化
- **扫描逻辑优化**：
  - 忽略 `Cleared` 和 `Backup` 目录，避免已通关游戏重复入库
  - 正确跳过已入库游戏（基于 metadata 修改时间）
- **字体渲染**：页面标题正确继承用户选择的字体
- **分类名称**：`unknown` 改名为 `Unclassified`
- **刮削过滤**：完善优惠码、解压密码、VIP 文本过滤

### 🐛 修复
- 修复 exe 名称为 `hgame_manager.exe`（原为 `jav_manager.exe`）
- 修复分页模式下搜索不到结果的问题
- 修复扫描时未正确跳过已入库游戏的问题
